LGPD

Proteção de dados pessoais — como a Renova Be cuida das suas informações.

A Lei Geral de Proteção de Dados (Lei nº 13.709/2018) regula o uso de dados pessoais no Brasil. A Renova Be está comprometida com a transparência e a segurança no tratamento das suas informações.

Quais dados coletamos

Para gestão do vínculo de trabalho, a empresa pode coletar dados como:

  • Nome, CPF, RG e dados de contato
  • Dados bancários para pagamento de salário
  • Informações de saúde (quando exigido pela legislação trabalhista)
  • Registros de acesso a sistemas internos
  • Histórico de treinamentos e avaliações de desempenho

Como usamos seus dados

Os dados são usados exclusivamente para:

  • Cumprimento de obrigações trabalhistas e legais
  • Gestão de benefícios e folha de pagamento
  • Comunicação interna e desenvolvimento profissional
  • Segurança de sistemas e instalações

Seus direitos

Como titular dos dados, você tem direito a:

  • Confirmar a existência de tratamento de seus dados
  • Acessar seus dados pessoais armazenados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a portabilidade dos dados
  • Ser informado sobre o compartilhamento de dados

Canal de privacidade

Para exercer seus direitos ou tirar dúvidas sobre proteção de dados, entre em contato com o DPO (Encarregado de Dados) pelo e-mail privacidade@renovabe.com.br.
